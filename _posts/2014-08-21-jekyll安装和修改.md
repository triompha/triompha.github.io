**2,在本机上安装jekyll环境**
修改gem源为taobao 
http://jekyllrb.com/

**3，修改评论提供和网站统计提供**
```
mkdir _include/custom
vim _include/custom/comments ：修改为多说提供的代码
vim _include/custom/analytics :修改为百度统计提供的代码

修改 _config.yml中为
comments :
    provider : custom
    
analytics :
    provider : custom
```
